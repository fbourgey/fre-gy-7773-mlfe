# ⏱️ Productivity Tools

- [Python documentation](https://docs.python.org/3/)

- Python libraries: [NumPy](https://numpy.org/), [pandas](https://pandas.pydata.org/), [SciPy](https://scipy.org/), [Matplotlib](https://matplotlib.org/).

- Learn some [Markdown](https://www.markdownguide.org/) (for README files, [Jupyter notebooks](https://jupyter-notebook.readthedocs.io/en/stable/), etc)

- Prefer the CLI (Command Line Interface) over GUI (Graphical User Interface) when possible.

- Jupyter Notebooks for interactive coding and documentation.

- Version control with [Git](https://git-scm.com/) and [GitHub](https://github.com/) for collaboration and code management.

- [VS Code](https://code.visualstudio.com/) for efficient coding and debugging (extensions, copilot, shortcuts)
  - Customize VS Code settings (see [here](https://code.visualstudio.com/docs/getstarted/settings))
  - Use keyboard shortcuts (see [here](https://code.visualstudio.com/docs/getstarted/keybindings))
  - Install useful extensions (see [here](https://code.visualstudio.com/docs/editor/extension-gallery))

- Leverage AI tools (e.g., GitHub Copilot, ask/edit/agent modes, etc.) to assist with coding, debugging, and learning!

- Use virtual environments (e.g., [venv](https://docs.python.org/3/library/venv.html), [conda](https://docs.conda.io/en/latest/)) to manage project dependencies.

- [ruff](https://docs.astral.sh/ruff/): extremely fast Python linter, formatter, and code fixer.

- [uv](https://docs.astral.sh/uv/): An extremely fast Python package and project manager, written in Rust.

- Write docstrings for functions and classes (see [PEP 257](https://peps.python.org/pep-0257/))
  - I use Numpy style docstrings (see [here](https://numpydoc.readthedocs.io/en/latest/format.html))
  - You may want to write type hints for function arguments and return types (see [PEP 484](https://peps.python.org/pep-0484/))

- Bonus:
  - Contribute to open source projects on GitHub.
    - Examples: [pandas](https://github.com/pandas-dev/pandas), [scikit-learn](https://github.com/scikit-learn/scikit-learn), [SciPy](https://github.com/scipy/scipy)
  - Write technical blogs or tutorials.
